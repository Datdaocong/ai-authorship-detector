# AI Authorship Detector

A lightweight machine learning project for detecting whether a piece of text was written by a human or generated by AI.

This project explores classical NLP techniques using **TF-IDF features and LinearSVC**, combined with a simple research pipeline and a demo web application.

---

# Project Overview

The goal of this project is to build a detector that can distinguish:

* **Human-written text**
* **AI-generated text**

The system uses:

* Character-level TF-IDF features
* Linear Support Vector Machine (LinearSVC)
* Real datasets merged from multiple sources

The project also includes:

* dataset pipeline
* experiment tracking
* error analysis
* demo web application

# Feature Experiments

Several feature configurations were tested.

| Feature Mode | Model     | CV Accuracy | Test Accuracy |
| ------------ | --------- | ----------- | ------------- |
| word n-grams | LinearSVC | 0.877       | 0.877         |
| char n-grams | LinearSVC | **0.9137**  | **0.9095**    |
| hybrid       | LinearSVC | 0.9137      | 0.9070        |

Character-level features performed best for this task.
